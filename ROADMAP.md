## Repositories

A repository class which provides you with simple CRUD operations like:

- `find`
- `findBy`
- `findOneBy`
- `findById`

- `create`
- `updateBy`

and many more. Get inspired by the [TypeORM](https://github.com/typeorm/typeorm) repository class.
